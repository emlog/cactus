# 仙人掌 AI 助手 (Cactus)

[English](./README.en.md)

仙人掌 AI 助手（Cactus）是一款专为 macOS 设计的智能文字处理工具。通过集成先进的 AI 技术，提供一键翻译、内容总结、单词查询及截图 OCR 识别等功能，极大提升您的办公与学习效率。**基于 MIT 协议完全免费且开源。**

---

### 核心功能
- **快捷翻译**：在任何地方选中文本，按下 `⌥ + X` 即可瞬间获取翻译结果。
- **智能总结**：快速提炼长篇文章或文档的要点，节省阅读时间。
- **上下文词典**：深入解释字词义项、词源，并提供丰富的用法示例。
- **截图 OCR**：支持截取屏幕区域进行文字识别，并自动完成翻译。
- **生词本与收藏**：轻松保存生词和翻译片段，支持基于遗忘曲线的复习计划。
- **权限引导优化**：更友好的辅助功能权限请求界面，支持在未授权时直接打开主窗口。
- **隐私保护**：您的数据保留在本地，仅与您配置的 AI 服务商进行交互。

### 安装说明

#### Homebrew (推荐)
通过 Homebrew 快速安装：
```bash
brew install emlog/cactus
```

#### 手动安装
1. 从 [GitHub Releases](https://github.com/emlog/cactus/releases) 页面下载最新版本。
2. 将 `Cactus.app` 拖入 `Applications`（应用程序）文件夹。

### 使用指南
- **默认快捷键**（均可在偏好设置中自定义）：
  - `⌥ + X`：翻译选中内容。
  - `⌥ + S`：总结选中内容。
  - `⌥ + Z`：字典查询。
  - `⌥ + A`：截图 OCR 翻译。
  - `⌥ + C`：打开主助手窗口。

### 配置说明
仙人掌 AI 助手支持多种主流 AI 服务商，如 OpenAI、DeepSeek、Claude、Gemini、智谱、火山引擎等。您只需在“偏好设置”中填入对应的 API Key 即可开始使用。

### ⚠️ 常见问题：提示“App 已损坏”或“无法打开”？
如果您在首次运行时看到 **“Cactus.app 已损坏，无法打开”** 或 **“身份不明的开发者”** 的提示，这是由于应用未经过苹果官方付费签名导致。

**解决方法：**
1. **执行终端命令**：
   打开 **终端 (Terminal.app)**，粘贴并运行以下命令（输入密码时不会显示字符）：
   ```bash
   sudo xattr -rd com.apple.quarantine /Applications/Cactus.app
   ```
2. **手动允许**（如果上述命令执行后仍无法打开）：
   前往 **系统设置 -> 隐私与安全性 -> 安全性** 模块，点击 **“仍要打开”**。
3. 现在您可以正常运行 Cactus 了。
