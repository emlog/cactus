仙人掌AI助手是一款基于大语言模型的智能文字助手，快速翻译、总结、解释选中文本。

### 繁体中文介绍

仙人掌 AI 助手是一款基於大型語言模型的智慧文字工具，能快速翻譯、摘要與解釋選取的文字內容。

主要功能：

• 快速翻譯：選取文字後，按下快捷鍵（預設 ⌥ + X），立即翻譯
• 智慧摘要：提煉重點內容，重點一目瞭然
• 白話解釋：以簡單易懂的語言說明複雜內容
